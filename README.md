# Machine Learning Portfolio 🚀🧠

Welcome to my comprehensive Machine Learning portfolio! This repository showcases a structured journey from essential data manipulation and visualization pipelines to advanced supervised ensemble networks, feature selection frameworks, and unsupervised clustering structures.

---

## 📂 Portfolio Structure & Case Studies

| # | Notebook File | Core Focus & Machine Learning Concepts | Key Libraries Used |
|---|---|---|---|
| 1 | `1_NumPy.ipynb` | Vectorized operations, array manipulations, and mathematical foundations for data pipelines. | `NumPy` |
| 2 | `2_Pandas_featureExtraction.ipynb` | Data wrangling, restructuring tables, missing value treatment, and custom feature extraction engineering. | `Pandas` |
| 3 | `3_data_visualization.ipynb` | Exploratory Data Analysis (EDA), distribution analysis, scatter matrices, and statistical plotting. | `Matplotlib`, `Seaborn` |
| 4 | `Loan_Logistic_Regression.ipynb` | Binary classification, probability thresholds, and performance metrics (Precision, Recall, ROC-AUC) for credit risk mapping. | `Scikit-Learn` |
| 5 | `Project_Insurance_ML.ipynb` | Continuous target prediction, multi-variable cost analysis, and evaluation metrics (MSE, $R^2$ score). | `Scikit-Learn` |
| 6 | `Project_IPL_Capstone.ipynb` | Comprehensive data processing, domain-specific feature engineering, and high-dimensional sports performance analysis. | `Pandas`, `Scikit-Learn` |
| 7 | `GridSearchCV,Randomized.ipynb` | Hyperparameter optimization configurations, tuning depth/learning rates, and cross-validation strategy design. | `Scikit-Learn` |
| 8 | `EnsembleLearning.ipynb` | Implementing high-performance predictive architectures using Random Forests, Boosting frameworks, and model stacking. | `XGBoost`, `Scikit-Learn` |
| 9 | `PCA_practice.ipynb` | Unsupervised dimensionality reduction, handling multicollinearity, and capturing maximum explained variance. | `Scikit-Learn` |
| 10 | `Unsupervised_Kclustering.ipynb` | Partition-based structural data segmentation, finding optimal k via the Elbow Method, and cluster centroid mapping. | `Scikit-Learn` |
